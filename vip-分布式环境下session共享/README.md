# 分布式架构下session共享及解决方案

> 分布式架构下，基于传统的session存储会话以及访问授权的方式已经无法适用了。对于这个问题如何解决？
业内已经提供了很多经过生产验证的解决方案，比如session复制、基于第三方存储中间件统一管理session、或者通过token机制来
做授权。本次课会基于这些解决方案做一个详细的说明。
