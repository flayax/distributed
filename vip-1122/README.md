# 分布式消息中间件-Kafka（2次课程）
> Kafka是一款分布式消息发布和订阅系统，具有高性能、高吞吐量的特点而被广泛应用与大数据传输场景。它是由LinkedIn公司开发，使用Scala语言编写，之后成为Apache基金会的一个顶级项目。
