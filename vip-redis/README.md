# 分布式缓存-Redis

> Redis是一个开源的、高性能的、基于键值对的缓存与存储系统，通过提供多种键值数据类型来适应不同场景下的缓存与存储需求; 最新版本已经更新到4.0.2了。本次课程讲解的是3.2.11版本

## 课程大纲

- **Redis数据结构及命令介绍和使用**

- **Redis实现简易的分布式锁**

- **Redis中Lua脚本的运用**

- **Redis持久化AOF/RDB**

- **Redis集群**

- **关于缓存击穿的思考及解决方案**
